# RuBrik — Living Blackboard Classroom Presentation

Open `index.html` directly in a browser. No backend or build process is required.

## Controls
- Next/previous: Arrow keys, Page Up/Page Down, Space, mouse wheel, touch swipe
- Home/End: first/last slide
- Escape: slide overview
- F: fullscreen
- M: mute/unmute classroom sounds
- N: presenter notes
- D: presenter chalk drawing mode
- E: eraser while drawing
- C: clear presenter drawing
- Ctrl+Z: undo drawing stroke
- 1–6: chalk color
- A: applause/confetti gimmick
- P: board capture mode
- Q: pop quiz overlay
- Konami code: detention mode

## Files
- `index.html`: 18-slide deck content
- `styles.css`: blackboard classroom visual system
- `script.js`: navigation, chalk draw helpers, interactions, confetti, drawing mode
- `design.md`: theme guide
- `rubrik.md`: source content

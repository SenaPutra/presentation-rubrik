# RuBrik HTML Presentation — Design System

## Theme

**The Living Blackboard Classroom**

The presentation feels like a passionate teacher explaining RuBrik through a giant interactive classroom blackboard.

Core ingredients:

* Dark green textured blackboard
* Colorful handwritten chalk
* Wooden board frame
* Teacher pointer animations
* Sticky notes and pinned papers
* Chalk diagrams and formulas
* Classroom bell transitions
* Pop quizzes
* Gold-star achievements
* Blackboard eraser transitions
* Colorful chalk confetti

Visual balance:

```text
60% Blackboard Classroom
20% Modern Product Presentation
10% Office Comedy
10% Interactive Web Experience
```

Main chalk palette:

```css
--chalk-white: #f4f1df;
```

Every slide is treated like a classroom lesson:

```text
LESSON 01
Masalah Informasi Internal
```

```text

Obrolan Warkop
```

```text

```

The final slide becomes a graduation moment:

```text
RuBrik
Informasi Sekarang Punya Rumah.

FINAL SCORE: READY TO DEPLOY
```

After clicking **AKTIFKAN RUBRIK**:

* Classroom bell rings
* Colored chalk explodes
* Gold stars appear
* Paper planes fly
* Massive chalk confetti launches
* Blackboard fills with celebratory doodles
* Final message appears:

```text
CLASS DISMISSED.
DEPLOYMENT BEGINS.
```
